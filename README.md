# Zephris

**日本株のバックテストを、ブラウザだけ・ノーコードで。**
*A browser-only, no-code backtesting tool for Japanese equities — honest by design.*

---

アクチュアリー出身、いまはデータサイエンス / AIの仕事をしています。

個人投資家として「この戦略は過去データで本当に有効だったのか」を確かめたくて、海外のツールをいくつか試しました。けれど「日本株を、ブラウザだけで、コードを書かずに」納得して検証できるものが見つからず、自分で作っています。それが **Zephris** です。

## Zephrisとは

本やネット、SNSで語られる投資の「通説」を、過去データに当てて確かめるための道具です。プログラミングは要りません。

- 戦略を組んで、過去の日本株データでバックテストする
- 市場平均や「ただ買って持つ」と並べて、勝ったのか負けたのかを見る
- 試した条件を残して、あとから同じ前提で開き直す

## 設計で大切にしていること

- **正直さ** — 勝った結果だけを見せない。あらかじめ決めたベンチマークを下回ったら、下回ったと出す。
- **BYOK / ローカルファースト** — 株価データもAPIキーも運営サーバーに置かない。各ユーザーのブラウザからJ-Quantsに直接つなぎ、結果は本人だけが見られる。
- **推奨しない** — 特定の銘柄や手法を勧めない。実在の銘柄は「検証対象」であって「推奨」ではない。
- **前提を管理する** — 手数料や税、ベンチマーク、後知恵（look-ahead / カーブフィッティング）を、最初から扱う。

バックテストは仮説の出発点で、未来を保証するものではありません。そこを正直に伝えながら、「確かめる」という行為を、もっと多くの人に開いていきたいと思っています。
